# Assets

Place the following image files here for production builds:

- `icon.png` — 1024×1024 app icon
- `adaptive-icon.png` — 1024×1024 Android adaptive icon foreground
- `splash.png` — splash screen image
- `favicon.png` — web favicon

These are referenced from `app.json`. For local development with Expo Go,
the app runs fine without them, but EAS builds will require real assets.
